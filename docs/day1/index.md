# Introduction to Bioinformatics

## Overview

Day 1 introduces the fundamental concepts of bioinformatics and their application to pathogen genomics and outbreak response.

---

## Learning Objectives

By the end of Day 1, participants will be able to:

- Explain the role of bioinformatics in infectious disease surveillance.
- Navigate the Linux command line.
- Access High Performance Computing (HPC) environments.
- Retrieve pathogen genomic data from public databases.
- Perform basic sequence data preprocessing.
